# Ivka & Marek - Wedding Website

Wedding website for Ivka & Marek  

## Live Site

https://m-holo.github.io/wddng/

## Setup

1. Clone this repository
2. Add your images to the `images/` folder:
   - `invitation.jpg` - Wedding invitation card
   - `couple-photo.jpg` - Couple photo
   - `photo1.jpg` - Gallery photo 1
   - `photo2.jpg` - Gallery photo 2
3. Update the Google Forms RSVP link in `index.html`
4. Update contact details in `index.html`
5. Push to GitHub

## Deployment

This site deploys automatically via GitHub Pages from the `main` branch.
